Coding

Personally, a few issues occurred during the creation of this webpage. Firstly, there was a struggle with placing the elements within a grid layout. Elements would be too small and not visible once the webpage was loaded, or it was too big and overlapped other elements once the webpage had loaded. Secondly, adjusting the webpages' dimensions to suit the mobile view, some images and elements had to be abandoned to create a perfect mobile web page. Additionally, It was challenging to find attractive colours. However, I can confidently say that I enjoyed creating this webpage and I look forward to developing my skills more.
