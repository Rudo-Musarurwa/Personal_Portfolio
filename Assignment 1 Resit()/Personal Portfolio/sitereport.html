Site Report

The development of this website was initially a bit challenging. However, through essential web development skills and further research, a full functioning website was achieved. The research was undertaken to see the expected format for a personal portfolio. A few ideas were found for the website design that could be used,some of the ideas were derived from the following page, shopify, then on a piece of paper, the frame and design were drawn, to visualise the design that was trying to be achieved.To get started, the contents of the pages were coding in HTML. Coding

A few issues occurred during the creation of this webpage. Firstly, there was a struggle with placing the elements within a grid layout. Elements would be too small and not visible once the webpage was loaded, or it was too big and overlapped other elements once the webpage had loaded. Secondly, adjusting the webpages' dimensions to suit the mobile view, some images and elements had to be abandoned to create a perfect mobile web page. Additionally, It was challenging to find attractive colours. However, I can confidently say that I enjoyed creating this webpage and I look forward to developing my skills more.
